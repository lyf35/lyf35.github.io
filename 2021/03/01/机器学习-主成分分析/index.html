机器学习-主成分分析

原理

主成分分析(Principle Components Analysis, PCA)属于无监督学习算法，它的想法是用一个qqq维的数据去尽可能地近似表达原本p(p>q)p(p>q)p(p>q)维的数据。

推导1：最小重建误差

我们将原始数据记作x1,x2,…,xN\boldsymbol{x}_1,\boldsymbol{x}_2,\dots,\boldsymbol{x}_Nx1​,x2​,…,xN​。假设现在构造了一个用来近似表示xi\boldsymbol{x}_ixi​的模型：f(λ)=μ+Vqλf(\boldsymbol{\lambda})=\mu + \boldsymbol{V}_{q}\boldsymbol{\lambda}f(λ)=μ+Vq​λ，其中μ\muμ是一个ppp维空间中的向量，Vq\boldsymbol{V}_qVq​是一个p×qp\times qp×q的矩阵，其中每一列为正交的单位向量，函数的输入参数λ\boldsymbol{\lambda}λ即为压缩后的qqq维向量。

f(λ)f(\boldsymbol{\lambda})f(λ)的目的是尽量地将压缩后的qqq维向量还原为ppp维向量，并且使得重构误差尽可能地小，用数学公式可以表达为：

min⁡μ,Vq,λi∑i=1N∣∣xi−μ−Vqλi∣∣2\min _{\mu,\boldsymbol{V}_q,{\boldsymbol{\lambda}_i}} \sum_{i=1}^{N}||\boldsymbol{x}_i-\mu -\boldsymbol{V}_q\boldsymbol{\lambda}_i||^2 μ,Vq​,λi​min​i=1∑N​∣∣xi​−μ−Vq​λi​∣∣2

通过对上式中的μ\muμ和λi\boldsymbol{\lambda}_iλi​求偏导，取偏导等于0作为极值点的条件，从中可求得：

μ^=xˉ\hat{\mu}=\bar{\boldsymbol{x}} μ^​=xˉ

λi^=VqT(xi−xˉ)\hat{\lambda _{i}}=\boldsymbol{V}_{q}^{T}(x_{i}-\bar{x}) λi​^​=VqT​(xi​−xˉ)

将上述结果带入待优化表达式，此时优化目标变为：

min⁡Vq∑i=1N∣∣(xi−xˉ)−VqVqT(xi−xˉ)∣∣2\min _{\boldsymbol{V}_q} \sum_{i=1}^{N}||(\boldsymbol{x}_i-\boldsymbol{\bar{x}})-\boldsymbol{V}_q\boldsymbol{V}_{q}^{T}(\boldsymbol{x}_{i}-\boldsymbol{\bar{x}})||^2 Vq​min​i=1∑N​∣∣(xi​−xˉ)−Vq​VqT​(xi​−xˉ)∣∣2

为了简便起见，令xˉ=0\boldsymbol{\bar{x}}=\boldsymbol{0}xˉ=0(如果不满足此条件可以对原始数据进行平移)。式中的VqVqT\boldsymbol{V}_q\boldsymbol{V}_{q}^{T}Vq​VqT​是一个p×pp\times pp×p的矩阵，它被称为投影矩阵，记作Hq\boldsymbol{H}_qHq​，它将每个xi\boldsymbol{x}_ixi​投影至它秩为qqq的重建向量上。

Vq\boldsymbol{V}_qVq​可以通过奇异值分解来计算。将所有中心化的xi\boldsymbol{x}_ixi​按行堆叠成一个N×pN\times pN×p的矩阵X\boldsymbol{X}X，通过将X\boldsymbol{X}X分解成X=UDVT\boldsymbol{X}=\boldsymbol{U}\boldsymbol{D}\boldsymbol{V}^TX=UDVT的形式，我们便可得到V\boldsymbol{V}V。

推导2：方差最大

PCA的另一种推导方式是从最大方差的角度去考虑，它的出发点是在数据降维之后，能够在降维后的特征上保持尽可能大的方差。这样在降维后的数据上，能够使得数据点尽可能地分散。

假设x=(x1,x2,…,xm)\boldsymbol{x}=(x_1,x_2,\dots,x_m)x=(x1​,x2​,…,xm​)是mmm维的随机变量，它的均值向量为μ=E(x)=(μ1,μ2,…,μm)\boldsymbol{\mu}=E(\boldsymbol{x})=(\mu_1,\mu_2,\dots,\mu_m)μ=E(x)=(μ1​,μ2​,…,μm​)，协方差矩阵为Σ=cov(x,x)=E[(x−μ)T(x−μ)]\Sigma=\text{cov}(\boldsymbol{x},\boldsymbol{x})=E[(\boldsymbol{x}-\boldsymbol{\mu})^T(\boldsymbol{x}-\boldsymbol{\mu})]Σ=cov(x,x)=E[(x−μ)T(x−μ)]。考虑由mmm维随机变量x\boldsymbol{x}x到mmm维随机变量y=(y1,y2,…,ym)\boldsymbol{y}=(y_1,y_2,\dots,y_m)y=(y1​,y2​,…,ym​)的线性变换：yi=xαiT=α1ix1+α2ix2+⋯+αmixmy_i=\boldsymbol{x}\alpha_i^T=\alpha_{1i}x_1+\alpha_{2i}x_2+\dots+\alpha_{mi}x_myi​=xαiT​=α1i​x1​+α2i​x2​+⋯+αmi​xm​，其中αi=(α1i,α2i,…,αmi),i=1,2,…,m\alpha_i=(\alpha_{1i},\alpha_{2i},\dots,\alpha_{mi}),i=1,2,\dots,mαi​=(α1i​,α2i​,…,αmi​),i=1,2,…,m.

如果上述的线性变换满足以下三个条件：

  1. 系数向量αi\alpha_iαi​为单位向量。这一条件表明线性变换是正交变换，{αi}\{\alpha_i\}{αi​}是一组标准正交基
  2. 变量yiy_iyi​与yjy_jyj​互不相关；
  3. 变量y1y_1y1​是x\boldsymbol{x}x的所有线性变换中方差最大的；y2y_2y2​是与y1y_1y1​不相关的x\boldsymbol{x}x的所有线性变换中方差最大的；以此类推，yiy_iyi​是与y1,y2,…,yi−1y_1,y_2,\dots,y_{i-1}y1​,y2​,…,yi−1​都不相关的x\boldsymbol{x}x的所有线性变换中方差最大的。

此时分别称y1,y2,…,yiy_1,y_2,\dots,y_iy1​,y2​,…,yi​为x\boldsymbol{x}x的第一、第二、……、第iii主成分。

条件2和3给出了计算主成分的方法：首先在x\boldsymbol{x}x的所有线性变换xα1T=∑i=1mαi1xi\boldsymbol{x}\alpha_1^T=\sum_{i=1}^{m}\alpha_{i1}x_ixα1T​=∑i=1m​αi1​xi​中，在α1α1T=1\alpha_{1}\alpha_{1}^{T}=1α1​α1T​=1的条件下，求出方差最大的，得到x\boldsymbol{x}x的第一主成分；然后在与xα1T\boldsymbol{x}\alpha_1^Txα1T​不相关的x\boldsymbol{x}x的所有线性变换中，xα2T=∑i=1mαi2xi\boldsymbol{x}\alpha_2^T=\sum_{i=1}^{m}\alpha_{i2}x_ixα2T​=∑i=1m​αi2​xi​中，在α2α2T=1\alpha_{2}\alpha_{2}^{T}=1α2​α2T​=1的条件下，求出方差最大的，得到x\boldsymbol{x}x的第二主成分；以此类推一直计算下去，可以得到x\boldsymbol{x}x的第mmm主成分。

首先我们求x\boldsymbol{x}x的第一主成分，即计算系数向量α1\alpha_1α1​。根据定义，第一主成分是在x\boldsymbol{x}x的所有线性变换xα1T=∑i=1mαi1xi\boldsymbol{x}\alpha_1^T=\sum_{i=1}^{m}\alpha_{i1}x_ixα1T​=∑i=1m​αi1​xi​中，在α1α1T=1\alpha_{1}\alpha_{1}^{T}=1α1​α1T​=1的条件下，使得方差var(xα1T)=α1Σα1T\text{var}(\boldsymbol{x}\alpha_1^T)=\alpha_1\Sigma \alpha_1^Tvar(xα1T​)=α1​Σα1T​最大的，也就是要求解下面的最优化问题：

max⁡α1 α1Σα1Ts.t. α1α1T=1\max_{\alpha_1}~\alpha_1\Sigma \alpha_1^T\\ s.t.~\alpha_{1}\alpha_{1}^{T}=1 α1​max​ α1​Σα1T​s.t. α1​α1T​=1

使用拉格朗日乘子法，可以定义拉格朗日函数α1Σα1T−λ(α1α1T−1)\alpha_1\Sigma \alpha_1^T-\lambda(\alpha_{1}\alpha_{1}^{T}-1)α1​Σα1T​−λ(α1​α1T​−1)，其中λ\lambdaλ为拉格朗日乘子。将拉格朗日函数对α1\alpha_1α1​求导，并令其为0，可得：

Σα1T−λα1T=0\Sigma \alpha_1^T-\lambda\alpha_1^T=\boldsymbol{0} Σα1T​−λα1T​=0

从上式可得，λ\lambdaλ为协方差矩阵Σ\SigmaΣ的特征值，α1\alpha_1α1​是特征值对应的单位特征向量。由于当α1\alpha_1α1​取最大特征值对应的特征向量时，α1Σα1T\alpha_1\Sigma \alpha_1^Tα1​Σα1T​取得最大值，此时α1Σα1T\alpha_1\Sigma \alpha_1^Tα1​Σα1T​的值为协方差矩阵的最大特征值λ1\lambda_1λ1​。

在求第二主成分时，我们需要加入额外的两个约束条件α1Σα2T=0\alpha_1\Sigma \alpha_2^T=0α1​Σα2T​=0和α2Σα1T=0\alpha_2\Sigma \alpha_1^T=0α2​Σα1T​=0。此时可求得第二主成分为协方差矩阵Σ\SigmaΣ的第二大特征值。以此类推，第kkk主成分即为协方差矩阵Σ\SigmaΣ的第kkk大特征值。

通过以上的推导可知，在对原始数据做降维时，我们只需要计算样本协方差矩阵的特征值和特征向量即可。在计算之前，需要将所有的样本中心化，即计算xi′=xi−xˉ\boldsymbol{x}_{i}'=\boldsymbol{x}_{i}-\boldsymbol{\bar{x}}xi′​=xi​−xˉ。在这一步完成之后，先计算所有特征的协方差矩阵C\boldsymbol{C}C，然后求出协方差矩阵的特征值λi\lambda _iλi​和特征向量vi\boldsymbol{v}_ivi​。通过对特征值进行排序，选出排名前kkk个特征值对应的特征向量。将原始的样本点投影到选出的这些特征向量上，便得到了降维后的特征。

通过协方差矩阵的特征值，我们也可以计算得到第kkk个主成分的方差贡献率为：ηk=λk∑i=1mλi\eta_{k}=\frac{\lambda_{k}}{\sum_{i=1}^{m}\lambda_{i}}ηk​=∑i=1m​λi​λk​​。

使用示例

Sklearn中的PCA类可以方便地完成主成分分析的操作，它的使用示例如下：

array([[-2.68412563,  0.31939725],
       [-2.71414169, -0.17700123],
       [-2.88899057, -0.14494943],
       ...,
       [ 1.76434572,  0.07885885],
       [ 1.90094161,  0.11662796],
       [ 1.39018886, -0.28266094]])

当数据量或者是特征数目很大的时候，无论是对特征矩阵计算协方差还是做奇异值分解，所需的内存以及计算量都很大。为了解决这一问题，有两种办法。

第一种办法是使用增量PCA的办法，这种办法允许我们将数据集分批输入，每次只输入一个小批量。这种办法对大型训练集很有用，而且可以用于在线学习的情景。Sklearn提供了IncrementalPCA类，用来做增量PCA，它的使用方法如下：

array([[ 2.56282819,  0.28217892],
       [ 2.41842883,  0.31240878],
       [ 1.94423122,  0.18790271],
       ...,
       [ 1.76430561,  0.07992847],
       [ 1.90016469,  0.12989783],
       [ 1.38953673, -0.2751793 ]])

另一种计算PCA的方法为随机PCA。它是一种随机算法，可以快速地找到前若干个主成分的近似值。这种算法的时间复杂度与选取的主成分个数有关，而不是与特征个数有关。因此当选取的主成分个数远小于特征个数时，它比普通的PCA算法快得多。

如果要使用随机PCA，则需要在PCA类初始化的时候传入参数svd_solver='randomized'：

array([[-2.68412563,  0.31939725],
       [-2.71414169, -0.17700123],
       [-2.88899057, -0.14494943],
       ...,
       [ 1.76434572,  0.07885885],
       [ 1.90094161,  0.11662796],
       [ 1.39018886, -0.28266094]])

附：奇异值分解

给定一个大小为m×nm\times nm×n的矩阵AAA，如果AAT=UΛ1UTAA^T=U\Lambda_1U^TAAT=UΛ1​UT，ATA=VΛ2VTA^TA=V\Lambda_2V^TATA=VΛ2​VT，则矩阵AAA的奇异值分解为：

A=UΣVTA=U\Sigma V^T A=UΣVT

矩阵U=(u1,u2,…,um)U=({\boldsymbol{u}_1,\boldsymbol{u}_2,\dots,\boldsymbol{u}_m})U=(u1​,u2​,…,um​)的大小为m×mm\times mm×m，其中ui\boldsymbol{u}_iui​为列向量，它们是矩阵AATAA^TAAT的特征向量，也被称为矩阵AAA的左奇异向量；矩阵V=(v1,v2,…,vn)V=({\boldsymbol{v}_1,\boldsymbol{v}_2,\dots,\boldsymbol{v}_n})V=(v1​,v2​,…,vn​)的大小为n×nn\times nn×n，其中vi\boldsymbol{v}_ivi​为列向量，它们是矩阵ATAA^TAATA的特征向量，也被称为矩阵AAA的右奇异向量；矩阵Λ1\Lambda_1Λ1​的大小为m×mm\times mm×m，矩阵Λ2\Lambda_2Λ2​的大小为n×nn\times nn×n，这两个矩阵对角线上的非零元素相同；矩阵Σ\SigmaΣ的大小为m×nm\times nm×n，位于对角线上的元素被称为奇异值。

设矩阵Λ1\Lambda_1Λ1​(或者Λ2\Lambda_2Λ2​)上的非零元素为λ1,λ2,…,λk\lambda_1,\lambda_2,\dots,\lambda_kλ1​,λ2​,…,λk​，其中kkk为矩阵AAA的秩；设矩阵Σ\SigmaΣ对角线上的非零元素分别为σ1,σ2,…,σk\sigma_1,\sigma_2,\dots,\sigma_kσ1​,σ2​,…,σk​(即奇异值)，则它们满足如下关系：

σ1=λ1,σ2=λ2,…,σk=λk\sigma_1=\sqrt{\lambda_1},\sigma_2=\sqrt{\lambda_2},\dots,\sigma_k=\sqrt{\lambda_k} σ1​=λ1​​,σ2​=λ2​​,…,σk​=λk​​

因此，奇异值分解的求解步骤可以分为下面三步：

  1. 求矩阵AATAA^TAAT的特征值和特征向量，用单位化的特征向量构成UUU
  2. 求矩阵ATAA^TAATA的特征值和特征向量，用单位化的特征向量构成VTV^TVT
  3. 将AATAA^TAAT或者ATAA^TAATA的特征值求平方根，然后构成Σ\SigmaΣ

需要注意的是，矩阵的奇异值分解结果中，奇异值是唯一的，但是矩阵PPP和QQQ不唯一。

奇异值分解的几何解释：矩阵UUU的列向量构成Rn\mathbb{R}^nRn空间的一组标准正交基，表示正交坐标系的旋转或反射变换；VVV的列向量构成Rm\mathbb{R}^mRm空间的一组标准正交基，同样表示正交坐标系的旋转或反射变换；Σ\SigmaΣ的对角元素是一组非负实数，表示RnR^nRn中的原始正交坐标系坐标轴的缩放变换。也就是说，原始空间的一组标准正交基，经过坐标系的旋转变换UUU，坐标轴的缩放变换Σ\SigmaΣ，坐标系的旋转变换VTV^TVT，得到了经过线性变换AAA等价的结果。可以用下图表示：
