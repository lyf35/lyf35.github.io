算法原理

朴素贝叶斯算法是基于贝叶斯定理和特征条件独立假设的分类方法。对于给定的训练数据集，算法会基于特征条件独立假设学习输入和输出的联合概率分布；然后对于给定的输入，利用贝叶斯定理求出后验概率最大的输出。这种算法实现简单，学习与预测效率都较高。

使用数学语言，朴素贝叶斯算法可以描述如下：

设输入空间X⊆Rn\mathcal{X}\subseteq R^nX⊆Rn为nnn维向量的集合，输出空间为类标记集合Y={c1,c2,…,cK}\mathcal{Y}=\{c_1,c_2,\dots,c_K\}Y={c1​,c2​,…,cK​}。输入为特征向量x∈X\boldsymbol{x}\in \mathcal{X}x∈X，输出为类标记y∈Yy\in \mathcal{Y}y∈Y。XXX是定义在输入空间X\mathcal{X}X上的随机向量，YYY是定义在输出空间Y\mathcal{Y}Y上的随机变量。P(X,Y)P(X,Y)P(X,Y)是XXX和YYY的联合概率分布，训练数据集D={(x1,y1),(x2,y2),…,(xm,ym)}D=\{(\boldsymbol{x}_1,y_1),(\boldsymbol{x}_2,y_2),\dots,(\boldsymbol{x}_m,y_m)\}D={(x1​,y1​),(x2​,y2​),…,(xm​,ym​)}由P(X,Y)P(X,Y)P(X,Y)独立同分布产生。朴素贝叶斯法通过训练数据集学习联合概率分布P(X,Y)P(X,Y)P(X,Y)。具体来说，联合概率分布P(X,Y)P(X,Y)P(X,Y)是通过计算先验概率分布P(Y=ck),k=1,2,…,KP(Y=c_k),k=1,2,\dots,KP(Y=ck​),k=1,2,…,K与条件概率分布P(X=x∣Y=ck)=P(X(1)=x(1),…,X(n)=x(n)∣Y=ck),k=1,2,…,KP(X=\boldsymbol{x}|Y=c_k)=P(X^{(1)}=x^{(1)},\dots,X^{(n)}=x^{(n)}|Y=c_k),k=1,2,\dots,KP(X=x∣Y=ck​)=P(X(1)=x(1),…,X(n)=x(n)∣Y=ck​),k=1,2,…,K来计算的。

计算机视觉-目标检测

发表于 2021-02-03 更新于 2021-02-07 分类于 理论知识 ， 计算机视觉 阅读次数：
本文字数： 13k 阅读时长 ≈ 11 分钟

概述

计算机视觉中关于图像识别有四大类任务：

  1. 分类-Classification：解决“是什么？”的问题，即给定一张图片或一段视频判断里面包含什么类别的目标。
  2. 定位-Location：解决“在哪里？”的问题，即定位出这个目标的的位置。
  3. 检测-Detection：解决“在哪里？是什么？”的问题，即定位出这个目标的位置并且知道目标物是什么。
  4. 分割-Segmentation：分为实例的分割（Instance-level）和场景分割（Scene-level），解决“每一个像素属于哪个目标物或场景”的问题。

目标检测（Object Detection）的任务是找出图像中所有感兴趣的目标（物体），并确定它们的类别和位置。所以目标检测是一个分类、回归问题的叠加。
