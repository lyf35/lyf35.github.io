论文笔记-Physics-informed Neural Network

发表于 2021-08-28 更新于 2021-09-05 分类于 论文笔记 阅读次数：
本文字数： 23k 阅读时长 ≈ 21 分钟

简介

本文汇总了Physics-informed neural networks: A deep learning framework for solving forward and inverse problems involving nonlinear partial differential equations和[Physics-informed machine learning](https://www.nature.com/articles/s42254-021-00314-5#:~:text=Physics-informed machine learning integrates seamlessly data and mathematical,regression methods offer effective%2C simple and meshless implementations.)这两篇文章中的主要思想。

在物理学、工程学等领域，经常会遇到数据难以获取的或者获取成本过高的情况，但是前沿的机器学习算法，尤其是神经网络，又无法保证小样本条件下的鲁棒性。考虑到一些实际问题通常需要满足某种物理规律或者经验公式，因此一个可行的办法是将这些模型引入到神经网络的训练过程中，作为网络的正则化项，从而将神经网络的输出限制到某个比较小的取值空间内，使用少量的数据也可以训练出一个预测较为准确的神经网络。而反过来说，将这些结构化的信息输入到模型中也相当于是对原始数据的信息增益，这样便使得模型可以接收到更多的信息，从而加速模型的收敛过程。

简介

卡尔曼滤波是一种根据线性系统状态方程，通过使用系统的输入输出与观测数据，对系统状态进行最优估计的算法。由于观测数据中包括系统中的噪声和干扰影响，所以最优估计也可以看做是滤波过程。简单地说，卡尔曼滤波要解决的是如何从多个满足高斯分布的不确定性信号中提取相对精确的信号。

论文笔记-PhysNet A Neural Network for Predicting Energies, Forces, Dipole Moments, and Partial Charges

发表于 2021-08-13 分类于 论文笔记 阅读次数：
本文字数： 6.2k 阅读时长 ≈ 6 分钟

简介

PhysNet: A Neural Network for Predicting Energies, Forces, Dipole Moments, and Partial Charges一文提出了PhysNet网络结构，可以用来预测分子的能量、偶极矩和原子的力等性质，它可以被归类到MPNN的图神经网络框架。这一网络结构在QM9、MD17和ISO17三个数据集上都取得了SOTA的结果。

此外，作者使用原子模拟的手段构造了两个新的数据集，用于衡量机器学习模型对于化学反应、长程作用关系和凝聚态的预测能力。其中一个是SN−2S_N-2SN​−2化学反应的数据集，PhysNet在这一数据集上的预测结果说明，在能量预测时加入电学性质能够使得模型预测的势能面在渐近区（Asymptotic region）更加准确。

作者也使用了一个包含一些小型肽键片段的Solvated protein fragments数据集去训练PhysNet，然后去预测分子量更大的蛋白质结构，并取得了几乎与第一性原理模拟相等的结果。

计算机视觉-风格迁移

发表于 2021-07-28 更新于 2021-08-01 分类于 理论知识 ， 计算机视觉 阅读次数：
本文字数： 12k 阅读时长 ≈ 11 分钟

概述

图像风格迁移是一种用不同风格渲染图像语义内容的图像处理方法，简单地说，就是提供一幅表示风格的图片，然后在尽可能保留图像内容的情况下，将照片转化为该风格。传统非参数的风格迁移方法只能通过提取图像的低层次特征（色彩、纹理等）来进行纹理合成，无法提取图像高层的特征。非参数风格的方法包括基于笔划的渲染、图像类比、图像滤波、纹理合成等。

近年来，随着深度神经网络在其它视觉感知领域（例如物体与人脸识别）的广泛应用，其在风格迁移领域也具有良好的表现。神经网络具有良好的特征提取能力，能够提取到丰富的语义信息，而这些信息正是风格迁移的基础。
