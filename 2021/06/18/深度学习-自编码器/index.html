深度学习-自编码器

发表于 2021-06-18 更新于 2021-07-11 分类于 理论知识 ， 深度学习 阅读次数：
本文字数： 14k 阅读时长 ≈ 13 分钟

概述

自编码器是神经网络的一种，经过训练后能尝试将输入复制到输出。这个网络可以看作由两部分组成：一个由函数h=f(x)\boldsymbol{h}=f(\boldsymbol{x})h=f(x)表示的编码器，和一个生成重构的解码器r=g(h)\boldsymbol{r}=g(\boldsymbol{h})r=g(h)。其中，f()f()f()和g()g()g()分别表示编码器和解码器所对应神经网络的函数表达式。

需要注意的是，如果一个自编码器只是简单地学会将处处设置为g(f(x))=xg(f(\boldsymbol{x}))=\boldsymbol{x}g(f(x))=x，那么这个自编码器就没有什么特别的用处，且不能被推广到新的实例。相反，通常是向自编码器强加一些约束，使它只能近似地复制，并且只能复制与训练数据相似的输入。这种约束强制模型考虑输入数据的哪些部分需要被优先复制，这样往往可以学到数据的有用特性。

目前，自编码器的应用主要包括降维、特征学习、去噪、图像生成、异常检测等。

欠完备自编码器

从自编码器中获得有用特征的一种方法是限制h\boldsymbol{h}h的维度比x\boldsymbol{x}x小，这种编码维度小于输入维度的自编码器称为欠完备自编码器。相应地，如果允许隐藏编码的维数与输入相等，或者隐藏编码维数大于输入，则被称为过完备自编码器，而在这种情况下，即使是线性编码器和线性解码器也可以学会将输入复制到输出，而学不到任何有关数据分布的有用信息。

学习欠完备的表示将强制自编码器捕捉到训练数据中最显著的特征。学习过程可以简单地描述为最小化如下的损失函数：

L(x,g(f(x)))L(\boldsymbol{x},g(f(\boldsymbol{x}))) L(x,g(f(x)))

其中LLL代指损失函数，用于惩罚g(f(x))g(f(\boldsymbol{x}))g(f(x))和x\boldsymbol{x}x的差异，如均方误差。

如果解码器是线性的，且LLL为均方误差，则欠完备自编码器会学习出与PCA相同的生成子空间。这种情况下，自编码器在训练去执行复制任务的时候，同时会学到训练数据的主元子空间。因此，拥有非线性编码器函数fff和非线性解码器函数ggg的自编码器能够学习出更强大的PCA非线性推广。但是，如果编码器和解码器的容量太大（或者说它们的参数太多从而导致近似能力太强），则自编码器仅仅会执行复制任务而捕捉不到任何有关数据分布的有用信息。

正则自编码器

概述

正则自编码器使用的损失函数鼓励模型除了将输入复制到输出之外，可以学习到一些其它特性，而不必限制使用浅层的编码器和解码器以及小的编码维数来限制模型的容量。即使模型容量大到足以学习一个无意义的恒等函数，非线性且过完备的正则自编码器仍然能够从数据中学到一些关于数据分布的有用信息。

下面介绍几种不同的正则自编码器。

稀疏自编码器

稀疏自编码器简单地在训练时结合编码层的稀疏惩罚Ω(h)\Omega(\boldsymbol{h})Ω(h)和重构误差：

L(x,g(f(x)))+Ω(h)L(\boldsymbol{x},g(f(\boldsymbol{x})))+\Omega(\boldsymbol{h}) L(x,g(f(x)))+Ω(h)

其中，惩罚项Ω(h)\Omega(\boldsymbol{h})Ω(h)可以被视为加到前馈网络的正则项。这里的惩罚项可以有多种选择，例如L1/L2正则化，KL散度等。

稀疏自编码器一般用来学习特征，以便用于像分类这样的任务。稀疏正则化的自编码器必须反映训练数据集的独特统计特征，而不是简单地充当恒等函数。

去噪自编码器

去噪自编码器（Denoising Autoencoder，DAE）是一类接受损坏数据作为输入，并训练来预测原始未被损坏数据作为输出的编码器。在去噪自编码器中，编码函数f(x)f(\boldsymbol{x})f(x)的概念被推广到编码分布pencoder(h∣x)p_{\text{encoder}}(\boldsymbol{h}|\boldsymbol{x})pencoder​(h∣x)。任何潜变量模型pmodel(h,x)p_{\text{model}}(\boldsymbol{h}, \boldsymbol{x})pmodel​(h,x)定义了一个随机编码器pencoder(h∣x)=pmodel(h∣x)p_{\text{encoder}}(\boldsymbol{h}|\boldsymbol{x})=p_{\text{model}}(\boldsymbol{h}|\boldsymbol{x})pencoder​(h∣x)=pmodel​(h∣x)和随机解码器pdecoder(x∣h)=pmodel(x∣h)p_{\text{decoder}}(\boldsymbol{x}|\boldsymbol{h})=p_{\text{model}}(\boldsymbol{x}|\boldsymbol{h})pdecoder​(x∣h)=pmodel​(x∣h)。

在DAE的训练过程中，需要引入一个损坏过程C(x~∣x)C(\tilde{\boldsymbol{x}}|\boldsymbol{x})C(x~∣x)，这个条件分布代表给定数据样本x\boldsymbol{x}x产生损坏样本x~\tilde{\boldsymbol{x}}x~的概率。自编码器按照下面的过程，从训练数据对(x~,x)(\tilde{\boldsymbol{x}},\boldsymbol{x})(x~,x)中学习一个重构分布preconstruct(x∣x~)p_{\text{reconstruct}}(\boldsymbol{x}|\boldsymbol{\tilde{x}})preconstruct​(x∣x~)：

  2. 从C(x~∣x)C(\tilde{\boldsymbol{x}}|\boldsymbol{x})C(x~∣x)中采集一个损坏样本x~\boldsymbol{\tilde{x}}x~，CCC通常为二项分布
  3. 将(x~,x)(\tilde{\boldsymbol{x}},\boldsymbol{x})(x~,x)作为训练样本来估计自编码器的重构分布preconstruct(x∣x~)=pdecoder(x∣h)p_{\text{reconstruct}}(\boldsymbol{x}|\boldsymbol{\tilde{x}})=p_{\text{decoder}}(\boldsymbol{x}|\boldsymbol{h})preconstruct​(x∣x~)=pdecoder​(x∣h)，其中h\boldsymbol{h}h为编码器f(x~)f(\boldsymbol{\tilde{x}})f(x~)的输出，pdecoderp_{\text{decoder}}pdecoder​根据解码函数g(h)g(\boldsymbol{h})g(h)定义
  4. 最小化损失函数LLL，损失函数可以为平方损失函数或者交叉熵。

收缩自编码器

另一个正则化自编码器的策略是使用一个类似于稀疏自编码器中的惩罚项Ω\OmegaΩ，此时损失函数为：

L(x,g(f(x)))+Ω(h,x)L(\boldsymbol{x},g(f(\boldsymbol{x})))+\Omega(\boldsymbol{h},\boldsymbol{x}) L(x,g(f(x)))+Ω(h,x)

其中，Ω(h,x)=λ∑i∣∣∇xhi∣∣2=λ∣∣∂f(x)∂x∣∣F2\Omega(\boldsymbol{h},\boldsymbol{x})=\lambda\sum_{i}||\nabla_\boldsymbol{x}\boldsymbol{h}_i||^2=\lambda ||\frac{\partial f(\boldsymbol{x})}{\partial \boldsymbol{x}}||^2_FΩ(h,x)=λ∑i​∣∣∇x​hi​∣∣2=λ∣∣∂x∂f(x)​∣∣F2​。FFF代表Frobenius范数。这迫使模型学习一个在x\boldsymbol{x}x变化小时，目标也没有太大变化的函数。因为这个惩罚只对训练数据适用，它迫使自编码器学习可以反映训练数据分布信息的特征。

收缩惩罚迫使自编码器学习到的所有映射对于输入的梯度都很小，即把输入都降维到一个很小的区域（点附近）；而重构误差迫使自编码器学习一个恒等映射，保留完整的信息。在这两种惩罚的作用下，使得大部分映射对于输入的梯度都很小，而只有少部分的大梯度。这样在输入具有小扰动时，小梯度会减小这些扰动，达到增加自编码器对输入附近小扰动的鲁棒性。

变分自编码器

原理

像标准自编码器一样，变分自编码器（Variational AutoEncoder，VAE）是一种由编码器和解码器组成的结构，经过训练以使编码解码后的数据与初始数据之间的重构误差最小。但是，为了引入隐空间的某些正则化，我们对编码-解码过程进行了一些修改：我们不是将输入编码为隐空间中的单个点，而是将其编码为隐空间中的概率分布。

VAE的模型训练过程如下：

  1. 使用编码器，将输入x\boldsymbol{x}x编码为隐空间上的分布q(z∣x)q(\boldsymbol{z}|\boldsymbol{x})q(z∣x)
  2. 从该分布中采样隐空间中的一个点z^∼q(z∣x)\hat{\boldsymbol{z}}\sim q(\boldsymbol{z}|\boldsymbol{x})z^∼q(z∣x)
  3. 使用解码器对采样点z^\hat{\boldsymbol{z}}z^进行解码，得到x\boldsymbol{x}x的重建x^\hat{\boldsymbol{x}}x^
  4. 计算重建误差，并通过反向传播更新网络参数

实践中，通常选择高斯分布作为编码的分布，这样网络的结构便如下图所示。其中，由于随机采样过程必须允许误差可以反向传播，因此随机采样这一步骤使用重参数化技巧进行表达：z=μ+σ⊙ϵ\boldsymbol{z}=\boldsymbol{\mu}+\boldsymbol{\sigma}\odot \boldsymbol{\epsilon}z=μ+σ⊙ϵ，ϵ∼N(0,I)\boldsymbol{\epsilon}\sim \mathcal{N}(0,\boldsymbol{I})ϵ∼N(0,I)。这样，均值和协方差向量便可以进行反向传播。

损失函数

根据贝叶斯定理，可得后验概率表达式：p(z∣x)=p(x∣z)p(z)p(x)p(\boldsymbol{z}|\boldsymbol{x})=\frac{p(\boldsymbol{x}|\boldsymbol{z})p(\boldsymbol{z})}{p(\boldsymbol{x})}p(z∣x)=p(x)p(x∣z)p(z)​，但是实际中p(x)p(\boldsymbol{x})p(x)无法计算，因而这个后验概率表达式不可解。

由全概率公式可得：p(x)=∫p(x∣z)p(z)dzp(\boldsymbol{x})=\int p(\boldsymbol{x}|\boldsymbol{z})p(\boldsymbol{z})d\boldsymbol{z}p(x)=∫p(x∣z)p(z)dz，而如果z\boldsymbol{z}z是一个维度很高的变量，则将变为一个多重积分式，难以计算。而变分自编码器的思想是使用一个可解的分布q(z∣x)q(\boldsymbol{z}|\boldsymbol{x})q(z∣x)去近似p(z∣x)p(\boldsymbol{z}|\boldsymbol{x})p(z∣x)，并希望二者的差距越小越好，也就是说二者的KL散度越小越好。因此，损失函数

L=min⁡KL(q(z∣x)∣∣p(z∣x))L=\min KL(q(\boldsymbol{z}|\boldsymbol{x})||p(\boldsymbol{z}|\boldsymbol{x})) L=minKL(q(z∣x)∣∣p(z∣x))

根据KL散度的定义，可得：

由于log⁡p(x)\log p(\boldsymbol{x})logp(x)是个定值，因此最小化KL(q(z∣x)∣∣p(z∣x))KL(q(\boldsymbol{z}|\boldsymbol{x})||p(\boldsymbol{z}|\boldsymbol{x}))KL(q(z∣x)∣∣p(z∣x))其实相当于计算如下表达式：

其中，第一项就指的是我们假设的后验分布q(z∣x)q(\boldsymbol{z}|\boldsymbol{x})q(z∣x)和先验分布p(z)p(\boldsymbol{z})p(z)尽量接近，而第二项则代表不断在z\boldsymbol{z}z上采样，然后使得被重构的样本中重构x\boldsymbol{x}x的概率最大。而其实第二项也可以看成是重建误差，这是因为解码器为一个固定结构，因此从z\boldsymbol{z}z到x^\hat{\boldsymbol{x}}x^的转换也是固定的，也就是说log⁡p(x∣z)\log p(\boldsymbol{x}|\boldsymbol{z})logp(x∣z)其实等价于log⁡p(x∣x^)\log p(\boldsymbol{x}|\hat{\boldsymbol{x}})logp(x∣x^)。
